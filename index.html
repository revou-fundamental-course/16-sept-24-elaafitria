Halo, saya Ela Afitria, seorang mahasiswa di Universitas Bengkulu, program studi Sistem Informasi, Fakultas Teknik. Saat ini, saya sedang mendalami berbagai aspek teknologi informasi, khususnya dalam pengembangan sistem yang dapat membantu memecahkan masalah di berbagai sektor. Sebagai seorang mahasiswa Sistem Informasi, saya tertarik untuk memahami bagaimana teknologi dapat diintegrasikan dengan kebutuhan bisnis dan masyarakat, serta bagaimana data dan informasi dapat dikelola untuk memberikan solusi yang efektif dan efisien. Selama masa studi, saya telah mengerjakan beberapa proyek, termasuk Sistem Monitoring Gizi Buruk Anak dan Sistem Informasi Manajemen Organisasi Mahasiswa, yang semakin mengasah keterampilan saya dalam pengembangan aplikasi dan sistem informasi.

Portofolio

Project 1

Sistem Informasi Monitoring Gizi Buruk Anak

Proyek ini merupakan platform yang dirancang untuk memantau status gizi anak-anak, khususnya di daerah yang rawan gizi buruk. Sistem ini memungkinkan petugas kesehatan untuk mencatat, memantau, dan menganalisis data terkait gizi anak-anak secara real-time.

Bahasa Pemrograman : PHP, MySQL, JavaScript

Project 1

Sistem Informasi Manajemen Organisasi Mahasiswa

Sistem ini dikembangkan untuk mempermudah pengelolaan administrasi dan kegiatan organisasi mahasiswa di berbagai universitas. Dengan sistem ini, pengurus organisasi dapat mengelola data anggota, kegiatan, hingga laporan keuangan secara lebih efisien. Selain itu, sistem ini juga memberikan akses bagi anggota untuk berpartisipasi dalam aktivitas organisasi melalui platform online.

Bahasa Pemrograman : PHP, MySQL, JavaScript
